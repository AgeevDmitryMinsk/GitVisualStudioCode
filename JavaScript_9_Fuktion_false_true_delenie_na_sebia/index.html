The function will return true if the number is greater than 1 and is divisible by only 1 and itself (that is, it is not divisible by any number between 1 and itself).

Example

isPrime(1) // false (Did you catch this case?) isPrime(13) // true isPrime(2) // true isPrime(14) // false isPrime(4) // false isPrime(22) // false
